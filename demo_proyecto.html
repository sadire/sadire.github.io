Simulador Botánico

Simulador diseñado para representar entornos agrícolas con precisión en tiempo real. Visualización de datos, sensores IoT y respuesta ambiental integrada.

Cada entorno simulado está basado en parámetros reales obtenidos por sensores de humedad, temperatura y pH, brindando un ecosistema visual coherente.

El sistema permite cambiar las condiciones del entorno de forma dinámica, generando respuestas visuales inmediatas.

Fue utilizado en entornos educativos y de investigación para probar modelos predictivos de crecimiento vegetal.
